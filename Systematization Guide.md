## 3️⃣.MAP 생성, 상품 적재, 주문내역 - 적재된 상품 LINK

**[구 조] **

* **선반 클래스 (SHELF):**

  A = SHELF()

  + 생성자 ------------------------- A()

  + 상품 적재 --------------------- A.push()

  + 상품 추출 --------------------- A.pop()

  + 선반에 상품 개수 ----------- A.stuffed()

  + 선반 시각화 ------------------ A.show()

    

* **상품 클래스(PRODUCT):**

  B = PRODUCT()

  + 생성자 ------------------------- B(code, name, ABC, x, y, z, g)

  + 상품 정보 출력 -------------- B

    

* **맵 클래스(MAP):**

  C = MAP()

  + 생성자 ------------------------- C()
  + i 번 선반에 물건 적재 ----- C.push()
  + i 번 선반에 물건 추출 ----- C.pop()
  + 물건 정보 --------------------- C.show_product(name)
  + i번 선반 물건 정보 출력 -- C.show_nth_shelf
  + 맵 정보 출력 ----------------- C
  + 맵 시각화 --------------------- C.show()
